An open, decentralized research platform built on the Sui stack that secures, indexes, and monetizes scientific knowledge using private AI agents

## 🚀 Quick Start

### Prerequisites

- **Python 3.8+**
- **pnpm 11+** for the frontend

### Running the Frontend

```bash
# 1. Clone and setup the project
git clone https://github.com/fullendmaestro/open-galaxy
cd apps/web
pnpm install
pnpm run dev
```

### Running the Agent

```bash
cd apps/agent
// create a venv and activate
pip install -r requirements.txt
// Install langgraph[---] cli
langgraph dev
```
